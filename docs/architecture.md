# Архітектура проєкту "Your Energy"

## Поточний стан

Проєкт розгортається на базі шаблону `vanilla-app-template`. Стан: **Початковий
етап (Initial Setup)**.

## Структура каталогів (планована)

- `src/` - вихідний код
  - `css/` - стилі компонентів (BEM)
  - `js/` - модулі JS (API, логіка)
  - `partials/` - HTML фрагменти
  - `img/` - зображення

## Ключові модулі (плановані)

- `main.js` - точка входу.
- `api.js` - взаємодія з бекендом (Your Energy API).
