# План реалізації проєкту "Your Energy"

## Етап 1: Підготовка та Базова структура (UI)

- [ ] **1.1. Структура та глобальні стилі:**
  - Перевірити відповідність папок правилам (`src/css`, `src/js`,
    `src/partials`).
  - Налаштувати `reset.css` та базові змінні (кольори, шрифти) у `base.css`
    згідно з макетом.
  - Підключити шрифти (Google Fonts).
- [ ] **1.2. Компоненти Header та Footer:**
  - Створити розмітку `src/partials/header.html` (лого, навігація, соцмережі).
  - Створити стилі `src/css/header.css` (адаптив, мобільне меню).
  - Створити розмітку `src/partials/footer.html` (форми підписки, контакти).
  - Створити стилі `src/css/footer.css`.
- [ ] **1.3. Головна сторінка (Home) - Статика:**
  - Створити Hero секцію (заголовок, теги).
  - Зверстати блок фільтрів (кнопки Muscles, Body parts, Equipment).
  - Підготувати контейнер для карток вправ.

## Етап 2: Логіка роботи з даними (JS Core)

- [ ] **2.1. API Сервіс:**
  - Створити `src/js/api.js`.
  - Реалізувати клас/модуль `YourEnergyAPI` з методами:
    - `getQuote()`
    - `getFilters(filter, page, limit)`
    - `getExercises(params)`
    - `getExerciseById(id)`
    - `patchRating(id, data)`
    - `postSubscription(email)`
  - Додати обробку помилок (try/catch) та базові логи.
- [ ] **2.2. LocalStorage Сервіс:**
  - Створити `src/js/storage.js`.
  - Реалізувати методи `save`, `load`, `remove` з безпечним парсингом JSON.

## Етап 3: Функціонал Головної сторінки

- [ ] **3.1. Цитата дня (Quote of the Day):**
  - Реалізувати логіку отримання цитати (перевірка дати в LocalStorage).
  - Відобразити цитату в UI.
- [ ] **3.2. Фільтри та Категорії:**
  - Реалізувати завантаження фільтрів (за замовчуванням "Muscles").
  - Рендер карток категорій.
  - Логіка перемикання вкладок фільтрів (оновлення списку категорій).
  - Пагінація для категорій.
- [ ] **3.3. Вправи (Exercises):**
  - Реалізувати клік по категорії -> запит вправ цієї категорії.
  - Приховати категорії, показати список вправ.
  - Рендер карток вправ (назва, рейтинг, кнопка Start).
  - Реалізувати кнопку "Back" (повернення до категорій).
- [ ] **3.4. Пошук:**
  - Реалізувати пошук вправ за ключовим словом (`keyword`).
  - Обробка події `submit` форми пошуку.
